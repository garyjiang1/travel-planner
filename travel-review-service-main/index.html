Developers

Up 
post /api/review
create a review of a trip (addReview)
Adds a review for a trip

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body Review (optional)
Body Parameter — Adds a review for a trip

Responses

201

item created

400

invalid input, object invalid

409

an existing item already exists
Up 
delete /api/review
Deletes Review (deleteReview)
By passing in the appropriate options, you can search for review to delete

Query parameters

user_id (required)
Query Parameter — pass a user id as search string for looking up review to delete
city (required)
Query Parameter — pass city as search string for looking up review to delete

Responses

201

review deleted

400

invalid input, object invalid
Up 
get /api/review
search for a review (getHealth)
By passing in the appropriate options, you can search for availalable review

Query parameters

city (required)
Query Parameter — pass a city as search string for looking up review
user_id (optional)
Query Parameter — pass an user id as search string for looking up review
skip (optional)
Query Parameter — number of records to skip for pagination format: int32
limit (optional)
Query Parameter — maximum number of records to return format: int32

Return type

array[Review]

Content-Type: application/json
[ {
